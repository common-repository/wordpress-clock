== Description ==

WordPress Clock provides you with a Flash Clock. You can use this functionality to display the local time.

* It could be used as a Widget (In your widget area)
* It provides you with a shortcode, which can be used anywhere on the website.
Shortcode: [ wpclock ]

Thanks to <a href="http://seomanageruk.com">SEO Manager UK</a>

== Installation ==

1. Download the zip file
2. Extract the contents
3. Upload the folder `wordpress-clock` to your WP plugin folder `/wp-content/plugins/` directory
4. Go to Plugins > Plugins, and activate the plugin

= Widget Installation =

1. Go to Appearance > Widgets, and drag the widget to a sidebar

= Shortcode Installation =

1. Use this shortcode '[wpclock]' on your page content to display the clock
